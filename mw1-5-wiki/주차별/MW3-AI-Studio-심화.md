# MW3 — AI Studio 심화

## 배운 것

3주차에서는 Google AI Studio를 깊이 사용하고, 웹사이트를 AI로 만드는 체험을 했습니다.

### 핵심 내용

- **AI Studio**: 구글의 AI 실험 환경. 프롬프트를 테스트하고 결과를 비교
- **시스템 프롬프트**: AI의 역할과 규칙을 미리 설정하는 방법
- **Stitch 시연**: AI가 웹사이트를 실시간으로 만드는 과정 관찰

### 시스템 프롬프트 예시

```
당신은 성우하이텍 제조팀의 품질 분석 전문가입니다.
불량 데이터를 받으면 원인을 분류하고 개선안을 제시합니다.
표 형식으로 정리해주세요.
```

### 기억할 포인트

| 개념 | 의미 |
|------|------|
| 시스템 프롬프트 | AI의 역할을 정해주는 설정 ([[Concept-CLAUDE-md]]와 비슷) |
| Temperature | AI 답변의 창의성 조절 (낮으면 정확, 높으면 창의적) |
| 토큰 | AI가 처리하는 텍스트 단위 (비용과 연결) |
