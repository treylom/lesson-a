# MW1 — AI란 무엇인가

## 배운 것

1주차에서는 AI의 기본 개념과 제조업에서의 활용 가능성을 배웠습니다.

### 핵심 내용

- **AI = 패턴을 찾아주는 도구**: 데이터에서 사람이 놓치는 패턴을 발견
- **생성형 AI**: 텍스트, 이미지, 코드를 만들어내는 AI (ChatGPT, Claude 등)
- **제조업 적용 사례**: 품질 검사 자동화, 불량 예측, 보고서 초안 생성

### 기억할 포인트

| 개념 | 의미 |
|------|------|
| LLM | 대규모 언어 모델 — AI가 말을 이해하는 방식 |
| 프롬프트 | AI에게 하는 질문이나 지시 ([[Concept-프롬프트]] 참조) |
| 할루시네이션 | AI가 사실이 아닌 내용을 만들어내는 현상 |

### 제조업 관점

> "AI가 금형 도면을 읽고 불량 가능성을 예측할 수 있다면, 
> 검사 시간을 크게 줄일 수 있습니다."
