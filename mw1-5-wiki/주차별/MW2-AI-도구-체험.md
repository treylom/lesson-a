# MW2 — AI 도구 체험

## 배운 것

2주차에서는 주요 AI 도구 3가지를 직접 사용해보고 차이를 체감했습니다.

### 체험한 도구

| 도구 | 특징 | 체험 내용 |
|------|------|----------|
| ChatGPT | 대화형, 범용 | 업무 질문 + 이미지 생성 |
| Gemini | 구글 연동, 검색 강점 | 최신 정보 검색 + 요약 |
| Claude | 긴 문서 처리, 정확성 | 보고서 분석 + 코드 이해 |

### 핵심 발견

- **도구마다 잘하는 게 다르다**: 만능 AI는 없음
- **같은 [[Concept-프롬프트|프롬프트]]도 도구마다 다른 결과**를 냄
- **업무에 맞는 도구 선택**이 중요

### 제조업 활용 비교

```
불량 원인 분석 → Claude (긴 데이터 처리)
시장 동향 검색 → Gemini (최신 정보)
아이디어 브레인스토밍 → ChatGPT (창의적 답변)
```
