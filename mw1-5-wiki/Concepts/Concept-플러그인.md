# 플러그인 — Claude Code에 기능을 추가하는 방법

## 한 줄 요약

플러그인은 Claude Code에 **새로운 능력을 추가하는 확장 도구**입니다.

## 쉬운 비유

스마트폰에 앱을 설치하는 것과 같습니다.
기본 기능만으로도 쓸 수 있지만, 앱을 깔면 더 많은 걸 할 수 있죠.

## 설치한 주요 플러그인

| 플러그인 | 하는 일 |
|----------|---------|
| superpowers | 체계적 사고, 브레인스토밍, 디버깅 도구 모음 |
| codex | 다른 AI와 토론시키기 |
| tofukyung-plugins | 강사가 만든 도구 모음 |

## 설치 방법

VSCode에서 Claude Code 채팅창에:
```
/plugins 열어줘
```
→ 마켓플레이스 URL 등록 → 원하는 플러그인 설치
