# student-profile.md — 나를 소개하는 파일

## 한 줄 요약

student-profile.md는 **내가 누구인지 AI에게 알려주는 파일**입니다.

## 쉬운 비유

병원에 가면 문진표를 쓰죠? "이름, 증상, 알레르기..."
student-profile.md는 AI에게 주는 문진표입니다.

## 포함하는 정보

| 항목 | 예시 |
|------|------|
| 이름 | 홍길동 |
| 부서 | 제조팀 (C그룹) |
| 역할 | 품질관리 담당 |
| 관심 업무 | 불량 패턴 분석, 공정 개선 |
| AI에게 바라는 것 | 데이터를 표로 정리, 쉬운 말로 설명 |

## 왜 중요한가

이 파일이 있으면 AI가:
- "제조팀이시니까 품질 관점에서 분석할게요"
- "금형 관련 용어를 써도 되겠네요"
- 매번 설명 안 해도 나를 기억
