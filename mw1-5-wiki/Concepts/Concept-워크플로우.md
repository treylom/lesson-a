# 워크플로우 — 반복 가능한 작업 순서

## 한 줄 요약

워크플로우는 **같은 작업을 매번 같은 순서로 반복할 수 있게 정리한 것**입니다.

## 쉬운 비유

제조 공정에서 "소재 투입 → 가공 → 검사 → 포장" 순서가 정해져 있죠?
AI 작업도 마찬가지로 순서를 정하면 매번 일정한 품질이 나옵니다.

## 5단계 루프 (MW5 Practice 08)

```
1. 데이터 준비 — 분석할 파일 선택
2. AI에게 지시 — 프롬프트 작성
3. 결과 확인 — AI 답변 검토
4. 수정 지시 — "이 부분 바꿔줘"
5. 저장 + 공유 — "저장해줘 → 올려줘"
```

## 실제 예시: 주간 품질 보고서

| 단계 | 행동 |
|------|------|
| 1 | 이번 주 불량 CSV 파일 준비 |
| 2 | "지난주 대비 변화 분석해줘" |
| 3 | 표와 그래프 확인 |
| 4 | "원인 가설도 추가해줘" |
| 5 | 저장 + Git push |

> 이 순서를 매주 반복하면, AI가 매번 비슷한 품질의 보고서를 만들어줍니다.
