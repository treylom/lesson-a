# Claude Code 공식 문서 요약

## 핵심 요약

Claude Code는 터미널에서 동작하는 AI 코딩 도구입니다.
파일을 직접 읽고, 수정하고, 실행할 수 있어서 반복 작업을 자동화할 수 있습니다.

## 주요 기능

| 기능 | 설명 |
|------|------|
| 파일 읽기/쓰기 | 프로젝트 폴더 안의 모든 파일을 직접 다룸 |
| 터미널 명령 실행 | git, npm 등 명령어를 대신 실행 |
| 웹 검색 | 최신 정보를 찾아서 정리 |
| 멀티 파일 수정 | 여러 파일을 한 번에 수정 가능 |

## 제조업 활용 포인트

- 보고서 초안 자동 생성
- 데이터 파일 분석 + 시각화
- 반복 업무 자동화 (매주 KPI 정리 등)
